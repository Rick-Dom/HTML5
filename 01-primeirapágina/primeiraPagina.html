CURSO DE HTML5


Introdução ao HTML

HTML (HyperText Markup Language) é uma linguagem de marcação, ela permite que você crie páginas web. ela foi desenvolvida na década de 90 por um cientista chamado Tim Berners-Lee que implementou a primeira comunicação bem sucedida entre um cliente http (navegador) e um servidor. Até chegar na versão que estamos hoje o html passou por diversas etapas. A W3C, que é um consórcio internacional também fundado por Lee onde visa desenvolver padrões e manter o crescimento da web, estava desenvolvendo uma versão do html chamada XHTML 2.0, mas um grupo de desenvolvedores de várias empresas montaram um grupo e começaram a desenvolver o HTML5. Apresentaram a proposta e por volta de 2006 a W3C largou de mão a outra versão e começou a trabalhar em conjunto com esse grupo e então surgiu a versão que conhecemos hoje com muitas mudanças boas.


obs: para criar sites você precisa de apenas duas coisas: um navegador e um editor de texto.

Criando a primeira página

Um documento HTML é constituído por tags e grande parte das vezes elas possuem uma tag de abertura e uma de fechamento, que são diferenciadas pelo sinal de /. Na primeira linha do nosso código definimos que estamos trabalhando com HTML5 com a tag !DOCTYPE html Em seguida vem a tag html que é a tag mais importante já que todas as outras entram nela. Podemos notar que as tags possuem atributos que são fundamentais para construção da página. A primeira que temos conhecimento é a lang que é inserida dentro da tag html e especifica a linguagem da nossa página.

Um documento html possui duas seções principais, onde uma vai carregar as informações sobre a página e a outra carregar o conteúdo da página, sendo respectivamente as tags head e body Uma coisa muito importante é lembrar sempre de manter tudo organizado para melhorar a visualização.

Dentro da primeira sessão nós podemos definir o título da nossa página com a tag title, juntamente com a tag que carrega o atributo charset que define qual conjunto de caracteres vamos utilizar
